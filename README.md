# Reda-WebTech-

This is a simple project I made for college to practice web technologies.

## How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/JoumJoumJoum/Reda-WebTech-.git
   cd Reda-WebTech-
   ```
2. Open the project in your code editor.
3. Run it (for example, open `index.html` in your browser).
